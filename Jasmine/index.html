Basic | Our journey through the objects

Create a simple object with the name car. This object should have properties: name, model, makeYear, color, fuelType. Create a method which will return the name and the model of the car.

Now update your code with the constructor function. Create a constructor function for the object car and create 5 instances (cars) of the same object.

Until now we had a method inside the object (after the last update our method should be in the constructor function). Now we are going to put the method which returns the name and the model of the car outside the constructor function using the prototype.

The last step in this exercise is to transform your constructor function into the class.
